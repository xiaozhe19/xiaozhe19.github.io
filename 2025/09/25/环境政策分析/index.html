环境政策分析导论

环境政策分析导论

  • 主体：直接间接制定政策的人（官方/非官方）
  • 客体： 政策发生作用的对象
    • ​ 社会问题和社会成员两个方面
  • 政策系统
    • 信息：神经系统 为政策制定 执行 评估 监控提供各种信息 作用体现在信息的收集 加工处理和传递
    • 咨询：重要组成部分 保证科学和民主
    • 决断：最终决策权力 主导公共决策活动的全过程
    • 执行：将政策方案转变为政策效果 为政策方案的执行做好准备（政策方案具体化、机构组织建设以及职权分配）
    • 监控：体制内和体制外的有关部门单位个人组成 作用为规范各个子系统的行为，监控反馈政策的执行情况
  • 政策分析的方法
    • 三个关键点的论证：1. 价值 是否实现（关键） 2. 事实 （的存在可能会影响价值的实现） 3. 行动（的采纳可能会影响价值的实现）
    • 经验的方法：它现在和将来会存在嘛？（事实）
      • 例子：
        在多个城市推广限塑令时，一位政策顾问建议：“我们以前在A市推行限塑令时，先从大型超市开始试点，效果很好。因此这次也应从大型商场先推行。”
    • 实证的方法：它有何价值？（价值）
      • 例子：
        研究人员收集多个城市限塑令实施前后塑料袋使用量的数据，并建立回归模型分析发现：限塑政策实施后塑料袋使用量平均减少了60%。
    • 规范的方法：应该干什么？（行动）
      • 例子：
        有人主张限塑令应该更严格，虽然成本较高，但更符合环保公平原则，“为了子孙后代的环境质量，我们应该更强制地限制一次性塑料制品。”
  • 环境政策的特点
    • 定义 国家（而非政府）为保护环境所采取的一系列控制 管理 调节措施的总和
    • 具体性：针对具体环境问题 where when + 环境要素
    • 费用有效性：更注重政策的费用效果和效率
    • 适时性：随着国家法律体系变化而变化
    • 多样性：环境问题的多样性和费用有效性决定了环境政策的多样性
  • 环境政策分析的目标
    • 分析存在问题
    • 对问题进行定性定量分析
    • 提供问题解决方案并提出环境改进建议
    • 便于学习和了解环境政策
  • 环境政策分析的特点
    • 以科学和技术的分析为基础
    • 观察视野的时空角度较大 环境问题有累积性 -> 需要更大的时间和空间范围才能充分体现
    • 信息需求量大但是往往难以满足
    • 不确定性大 分析可靠性差 强调及时性 -> 基于政策分析的政策选择风险大
    • 分析结论要简练 便于干系人进行政策交流
  • 环境政策分析存在的问题：环境政策分析的研究领域、理论视角、核心议题与方法论； 这些书上没有

理论基础

  • 前置问题

    • 环境政策分析主要研究 协调环境资源利用中的个体决策冲突问题 -> 外部性问题 外部性 = 个体理性 vs 集体不理性
    • 个体决策在市场或者计划等级制内冲突的程度 -> 环境问题的严重程度
    • 环境政策：个体决策在环境资源利用过程中的相互妥协的安排

  • 传统外部性理论

    • 外部效应： 某个体的决策对其他个体的收益（成本）造成影响

    • 缺少市场：没有市场，没有相应的支付发生 -> 价格信号失真

    • 外部性： 指现实发生的一种损失，这种损失发生在利益相关者之间，已被识别，并且足够采取一种可行方案，来使利益相关者的这种损失减少。

      • ✅ “现实发生的一种损失”

        意思是：外部性并不是一个抽象的可能性，而是已经真实存在的问题和损害。

        • 举例：工厂排污导致河流污染，这是已经造成损失，不是潜在的风险。

        ✅ “发生在利益相关者之间”

        这里强调：外部性不是孤立存在的，而是因为某些人的行为影响了其他人。

        • 工厂排污 ➜ 下游居民不能用水 ➜ 工厂和居民就是利益相关者。

        ✅ “已被识别”

        意味着：这个损失是可以观察、测量和证明的，而不是模糊的猜测。

        • 检测发现水体中污染物超标，医院数据表明居民生病率上升 ➜ 外部性已经被识别。

        ✅ “足够采取一种可行方案”

        说明这个问题已经严重到足以引起干预，而且是可以被政策手段解决的。

        • 比如可以安装污水处理设备、建立补偿机制、实施排放标准等。

      • 定义强调四个方面

        • 外部性是一种增加社会福利的机会

        • 是相互的 从总体比个体更容易把握住

        • 外部性识别需要信息 信息和相关知识是外部性的政策分析的前提

        • 外部性内部化（“你造成的损失，自己买单；你带来的好处，应该被奖励。”）成本很高

          • 🎯 工厂排污 ➜ 内部化办法：

            外部性情形（未内部化） 内部化后的办法
            工厂把废水直接排进河 ➜ 自己不承担后果 政府对每吨污水收排污费（排污税）
            下游村民农田污染 ➜ 工厂不赔偿 村民起诉企业 ➜ 法院判赔偿
            污染问题社会承担 ➜ 医疗、清理费用由公众买单 政府立法要求企业建污水处理设施

            ✅ 上面这些手段的目的，就是把“原来社会承担的污染代价”，变成企业自身的成本，让它自己考虑环保问题。

      • 产权 交易费用 信息不对成 共同解释外部性的原因

        • 产权 外部性的生命周期是一个产权逐渐清晰的过程

        • 交易费用 界定产权以及保护产权的成本

        • 主要原因是克服信息不对成

        • 🌀 初期阶段（产权极度模糊、外部性不被关注）：

          • 十几年前，一个乡镇新建了一家工厂，它每天排放废气，但村民觉得“能带来就业”，没当回事。
          • 这时产权很模糊：空气是谁的？污染算谁的责任？没人关心。
          • ➜ 外部性存在，但没有人尝试治理。

          🔍 中期阶段（外部性显现、开始界定产权）：

          • 几年后，村民发现家里灰尘多、小孩咳嗽多，才意识到污染有害。
          • 他们上访、请环保部门调查。经过一番努力，政府开始立法限制排放。
          • ➜ 这时候产权开始清晰（比如政府说“空气使用权属于公众，污染要限排”），
            但过程需要投入检测设备、调查费用、制定法律 ➜ 就是“交易费用”。
          • 同时也发现很多污染源难以追溯 ➜ 这体现了“信息不对称”。

          ✅ 后期阶段（产权清晰、外部性被内部化）：

          • 环保标准建立，排放要申请许可证、多排要交罚款 ➜ 外部性被“内部化”了。
          • 空气的“产权”虽然仍是公共的，但已通过制度安排让企业“为污染付出代价”。

    • 环境外部性理论

      • 外部性的主体

        • 作用主体 谁影响了谁
        • 责任主体 谁有这样影响的权力

      • 影响范围

        • 时间尺度 代际 代内 10-25为一代
        • 行政区 市内 跨市 跨省 跨国 全球

      • 外部性的大小

        • 边际社会成本和边际私人成本的差额
        • 包含外部损失 外部收益 内部化成本 内部化收益
          • 净大小 外部收益-外部损失 环境一旦发生就存在
          • 内部化净效益大小 内部化收益-内部化成本 采取某种措施以防止一定程度的污染所需的额外成本和带来的收益的差距

      • 外部性测量方法

        • 专家评估
        • 讨价还价 通过建立干系人的协商机制，让获得了信息的主体为了得到最大的私人利益，主动披露和调整自己的支付意愿和受偿意愿 基于讨价还价的过程和结果所传递的信号 获得对外部性大小的认识的过程
          • 这个认识更加感性和直接 经过实践检验 认可度高 但是成本大 受到影响较大

    • 外部性视角下环境政策分析

      • 政府与市场的分工 小政府与大市场 明确产权
        • 法律法规的完善 -> 产权的界定和调整
        • 执行 -> 产权的保护
        • 干预市场 -> 产权的交易
        • 政府干预承担公众的代理人 支付公众监督的成本和寻租的成本
        • 作为委托人 企业为代理人 需要支付政府监督的成本和企业违法的成本
        • 通过公众参与来降低处理环境问题的成本 以民为本 服务型政府
      • 对象体系 完备性与差别性
        • 完备性 政策的作用对象囊括所有的干系人
        • 干系人决策成本不同 监督的难易程度不同 需要区别对待
      • 环境管理体制：中央与政府的分工
        • 中央： 制定全国环境管理法规 直接管理跨省外部性问题
        • 省： 跨市外部性问题

    • 环境管理的决策者：指的是有权力、有责任对环境问题作出决策和采取行动的人或机构 政府 企业管理层 ngo 公众和媒体

    • 环境管理的决策议题的提出 由下而上：公众或媒体反映问题 → 政府介入 由上而下：政府或科研机构主动发现问题（如气候监测）由外部压力推动：国际组织、外企或突发事件（如环境事故）

    • 科学管理的决策程序

环境政策手段

  • 环境政策手段 政府为控制或消除社会造成负面环境影响所采取的措施

  • 分类

    • 命令控制类
    • 经济刺激类
    • 劝说鼓励类

  • 评价标准 政策目标的实现程度

    • 确定性标准

      • 控制的程度 政策对对象产生的影响 对政策目标实现多少 越高 确定性越强
      • 见效时间 政策见效的时间

    • 经济效率标准

      • 建立再费用效益分析基础之上的 单位成本所能产生最大价值 单位价值所需要的最小成本

    • 持续改进标准

      • 衡量环境政策手段产生的深化程度 能否对个人以及组织产生的持续刺激 促进他们不断降低环境损害

    • 公平性标准

      • 考虑各项环境政策手段的实施在不同主体间产生的分配效果

  • 命令控制手段

    • 直接规定或者命令来限制污染物排放 -> 标准

    • 有明确的法律依据 权威性和强制性

    • 严格的处罚措施

    • 较强的政策执行能力

    • 适用的领域

      • 适用范围最广 不可或缺 主导地位

    • 我国的命令控制政策分析

      • 分类
        • 事前
        • 事中
        • 事后

    • 分析示例

  • 经济刺激手段

    • 只要当某种手段的应用足以影响到经济当事人对可选择行动的成本进行评估的手段

    • 具有刺激性而非强制性

    • 定义：政府管理当局从影响成本-效益入手，影响经济当事人进行选择，以便于最终有利于环境改善的政策手段

    • 类型

      • 创建市场 科斯手段 产权/分散权力 可交易的许可证 国际补偿制度
      • 利用市场 庇古手段 补贴/减少补贴 环境税 排污收费 押金-退还制度 专项补贴

    • 原理

      • 依靠政府机构的法定职能和权威直接给政策对象提供经济刺激 基于职权的
      • 改变市场信号 依靠市场机制简介给对象刺激 基于市场的

    • 特点

      • 足够的刺激
      • 政府和市场的规范
      • 良好的监测技术

    • 适用领域

      • 污染控制（为主） 生态保护

  • 劝说鼓励手段

    • 定义 基于意识转变和道德规劝影响人们环境保护行为的政策手段

    • 原理 最终目的 强化被管理者的环境意识 自觉地去保护环境

    • 特点

      • 强制性若
      • 强调预防性
      • 制定成本和执行成本较低
      • 长期效果好

    • 适用领域 大量发生 较为分散的环境问题 紧急的环境问题不行 不能滥用 逆反心理

  • 环境政策手段组合和设计：原则与方法

    • 原则

      • 历史的经验 不能盲目照搬 要与历史条件和现有条件对照之后修正 应用于本地
      • 政策的目标
      • 实施的条件

    • 方法

      • 调查具体情况
      • 明晰政策目标
      • 形成环境政策手段选择框
      • 分析选择出组合矩阵
